Graph invariant

Graph invariant

Jan 12, 20261 min read

PAL Definice Invariant grafu

Function over a graph, for which following holds: Examples

  • number of node
  • number of vertices
  • sum of unique degrees (both in and out), sorted
  • sequence of degrees on a path

If invariant doesn’t hold, graphs are not isomorphic.
